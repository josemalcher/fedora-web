Tutorial: Gnote

  1. To create your own personal wiki using Ç´note. Start Gnote by clicking Applications > Accessories. Then you must click "Create New Note" Now enter what you want on this new note. Notes in Gnote can be linked together by highlighting text in the current note and clicking the Link button above in the toolbar. Doing so will create a new note and also underline the note's title in the current note.

  2. Changing the title of a note will update links present in other notes. This prevents broken links from occurring when a note is renamed.

  3. Also, if you type the name of another note in your current note, it will automatically be linked for you.
