Fedora 8 Release Notes

Here are places you can find the Fedora 8 release notes. If one location is unresponsive, try a different one.

  • http://docs.fedoraproject.org/release-notes/
  • http://fedoraproject.org/release-notes/

Check any mirror using a path similar to http://URL/fedora/linux/releases/8/Fedora/i386/os/. The mirros only carry the Enhlish version, RELEASE-NOTES-en_US.html. Note that each mirror may have a slightly different path, but should be consistent below the 8/ directory. For example:

  • RPMFind.net mirror
