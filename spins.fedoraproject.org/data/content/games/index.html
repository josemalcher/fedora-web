Installable Live Media

This is the latest version of the Fedora Linux operating system's Games spin. It's everything you need to try out Fedora's games spin--you don't have to erase anything on your current system to try it out, and it won't put your files at risk. Take the Fedora Games spin for a test drive, and if you like it, you can install it directly to your hard drive straight from the Live Media desktop if you like.
