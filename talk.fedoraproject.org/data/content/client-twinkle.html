${_('Configuring Twinkle for Fedora Talk')}

${Markup(_('There is a <a href="http://fedorapeople.org/groups/web/fedoratalk-twinkle-screencast.ogg">useful video tutorial</a> for configuring Twinkle with Fedora Talk. Useful content may be added to the <a href="http://fedoraproject.org/wiki">Fedora Project wiki</a>, possibly even by <em>you</em>.'))}

  1. ${_('Install the twinkle package by running the following command:')}
    su -c 'yum install twinkle'

    2. Or use PackageKit

    1. From the main menu, select System → Administration → Add/Remove Software
    2. Type "twinkle" into the "find" box and click find
    3. Check the box for the "twinkle" package
    4. Click "Apply"
  2. Select Applications → Internet → Twinkle from the main menu
  3. Select "Ok" to create a profile

    4. Recommend the profile editor as it's pretty simple, but the flow here shows the wizard :)

  4. Select "Wizard"

  5. Select "Profile Editor"

  6. Give the profile a name, such as "FedoraTalk" (spaces are not allowed)

${_('Wizard Workflow')}

  1. Fill in the following information:
    • SIP service provider: Other
    • Your Name (optional): Firstname Lastname
    • User Name: FAS Account name
    • Domain: talk.fedoraproject.org
    • Authentication Name: FAS Account name (If you want password saved locally)
    • Password: VOIP Password (this can be found in your fas account and is NOT your FAS account password) Choose OK
  2. Click OK
  3. A warning will appear indicating that system settings can be adjusted. Choose OK
  4. Select the "SIP Server" icon in left column and fill in the following information:
    • Registrar: talk.fedoraproject.org (This defaults from the domain so doesn't need to be entered.)

${_('Profile Editor Workflow')}

  1. USER Fill in the following information:
    • SIP Account
    • Your Name (optional): Firstname Lastname
    • User Name: FAS Account name
    • Domain: talk.fedoraproject.org
  2. SIP Authentication
    • Realm (optional): LEAVE BLANK
    • Authentication name: FAS Account name
    • Password: VOIP Password (this can be found in your fas account and is NOT your FAS account password)
  3. SIP SERVER Fill in the following information
    • Registrar: talk.fedoraproject.org (You can leave this blank.)
    • Choose OK
  4. To receive a call you must be registered, this should happen automatically if the above settings are correct.
    • To verify your registration click on "Registration/Show Registrations"
    • your firewall should allow inbound connections on port 5060
  5. Verify that your audio setup is working by calling the Echo Test line on the Fedora Talk system
